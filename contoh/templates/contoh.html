Contoh soal 1

Besar sudut BAC adalah...

JAWAB :

2x + 4° + 8x + 1° + 4x + 7° = 180°

14x + 12° = 180°

x = 12°

Sehingga besar sudut BAC

2x + 4° = 2(12) + 4° = 28°


Contoh soal 2

Perhatikan gambar ! Apabila sudut ABD adalah siku-siku, besar sudut CBD adalah ...

∠ABC + ∠CBD = 90°

65° + ∠CBD = 90°

∠CBD = 90° - 65°

∠CBD = 25°

Jadi, besar ∠CBD = 25°


Contoh soal 3

Perhatikan gambar!

Nilai y pada gambar di atas adalah ....

JAWAB :

3x = 180° - 120° (sehadap dengan suplemen sudut 120°)

3x = 60°

2y = 3x (bertolak belakang)

2y = 60°

y = 30°

Jadi, nilai y adalah 30°
